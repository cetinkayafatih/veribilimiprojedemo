Olympic Insights Lab

Data Science Demo - Olympics Analysis

Istanbul Bilgi University | Fall 2024-2025

Project Overview

This project analyzes Olympic Games data to uncover insights about gender parity, national medal performance, and athlete success factors. Using data from 1896 to 2024, we apply statistical and machine learning techniques to answer three key research questions.

Research Questions

Q1: Gender Parity

Descriptive Analysis

How has the gender participation gap evolved from 1896 to 2016, and what is the discipline-level parity at Paris 2024?

Q2: Medal Prediction

Multiple Linear Regression

Can we predict a country's medal count based on delegation size and historical performance?

Q3: Athlete Classification

Logistic Regression

Do biometric features (age, height, weight) predict medal success in high-physicality sports?
